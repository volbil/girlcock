## Розгортання

Для старту необхідно встановити залежності:

```
uv sync
```

Далі необхідно створити файл `settings.toml` з токеном бота:

```
[default]
    [default.bot]
    token = "TOKEN"
```

І запуск самого бота:

```
uv run bot.py
```
